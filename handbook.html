1. Iniziare

Registrazione dell'Account

Per iniziare a utilizzare ChatBot Hub, segui questi passaggi:

  1. Clicca su "Registrati" nella pagina principale
  2. Inserisci il tuo nome, email e una password (minimo 8 caratteri)
  3. Completa il processo di registrazione
  4. Verifica il tuo account tramite il codice OTP inviato via email

Processo di Login

  1. Clicca su "Accedi" nella pagina principale
  2. Inserisci la tua email e password
  3. Clicca "Accedi" per entrare nella tua dashboard

Comprendere la Dashboard

La dashboard è il tuo centro di controllo principale. Qui puoi:

  • Visualizzare le informazioni del tuo account
  • Monitorare l'utilizzo dei messaggi e delle immagini
  • Gestire i tuoi piani e sottoscrizioni
  • Accedere alle impostazioni API

2. Configurazione API

Come ottenere una OpenAI API Key

  1. Vai su https://platform.openai.com
  2. Accedi o registrati al tuo account OpenAI
  3. Vai alla sezione "API Keys" nel tuo profilo
  4. Clicca su "Create new secret key"
  5. Copia la chiave generata (inizia con "sk-")
Nota: La tua API Key viene salvata solo nel browser locale (localStorage) e non viene mai inviata al server. Questo garantisce la massima sicurezza.

Come salvare la API Key nell'app

  1. Vai nella sezione "Configurazione" dalla navbar
  2. Incollare la tua API Key OpenAI nel campo "OpenAI API Key"
  3. Clicca "Salva" per memorizzare la chiave

Risoluzione problemi di connessione

  • Verifica che la tua API Key sia corretta
  • Assicurati di avere credito sufficiente sul tuo account OpenAI
  • Controlla la tua connessione internet
  • Prova a ricaricare la pagina o a cancellare la cache del browser

3. Creare e Gestire Chatbots

Guida passo-passo per creare un nuovo chatbot

  1. Vai alla sezione "I Miei Bot" dalla navbar
  2. Clicca sul pulsante "+ Nuovo Bot"
  3. Compila il modulo di creazione:
    • Nome Bot: Scegli un nome descrittivo (max 100 caratteri)
    • Personalità/Ruolo: Definisci il comportamento e le competenze del bot (max 5000 caratteri)
    • Modello: Seleziona il modello AI (GPT-4o-mini o GPT-4.1-nano)
  4. Clicca "Crea Bot" per finalizzare

Scegliere nomi e personalità dei bot

Nomi efficaci: "Assistente Cucina", "Tutor Matematica", "Consulente Marketing"

Personalità efficaci: "Sei un esperto chef italiano specializzato in cucina regionale..."

Selezionare i modelli AI

  • GPT-4o-mini: Modello più leggero, adatto per conversazioni generali
  • GPT-4.1-nano: Modello più potente, adatto per compiti complessi

Modificare le configurazioni dei bot

Attualmente, per modificare un bot, è necessario eliminarlo e crearne uno nuovo con le impostazioni aggiornate.

Eliminare i chatbots

  1. Vai alla sezione "I Miei Bot"
  2. Trova il bot che vuoi eliminare
  3. Clicca sul pulsante "Elimina"
  4. Conferma l'eliminazione

4. Chattare con i Bots

Avviare conversazioni

  1. Vai alla sezione "I Miei Bot"
  2. Clicca "Apri Chat" sul bot con cui vuoi parlare
  3. Scrivi il tuo messaggio nell'area di input
  4. Clicca "Invia" o premi Enter

Inviare messaggi e comprendere le risposte

I messaggi vengono visualizzati in tempo reale. I tuoi messaggi appaiono a destra, le risposte del bot a sinistra.

Memoria del contesto (finestra di 50 messaggi)

Il sistema mantiene una memoria di contesto di 50 messaggi per garantire conversazioni coerenti. I messaggi più vecchi vengono automaticamente rimossi quando si supera questo limite.

Panoramica della cronologia delle conversazioni

Tutte le conversazioni vengono salvate e sono accessibili quando apri un bot. Puoi scorrere verso l'alto per rivedere la cronologia completa.

Consigli per risposte migliori

  • Sii specifico nelle tue richieste
  • Fornisci contesto quando necessario
  • Usa frasi complete e grammatica corretta
  • Sii paziente per risposte complesse

5. Generazione Immagini

Utilizzare la funzione di generazione immagini

  1. Apri una chat con un bot
  2. Scrivi una descrizione dettagliata dell'immagine che vuoi generare
  3. Clicca sul pulsante "Immagine" invece di "Invia"
  4. Attendi che l'immagine venga generata e visualizzata

Limiti giornalieri di immagini per piano

Piano Immagini/giorno
Free 3
Basic 10
Premium Illimitate

Consigli per prompt efficaci

  • Sii specifico: "Un tramonto tropicale con palme e mare turchese"
  • Specifica lo stile: "Stile pittorico, acquerello"
  • Includi dettagli: "Luce dorata, ombre lunghe, cielo con nuvole"
  • Evita ambiguità: "Vista frontale di un gatto bianco con occhi blu"

Visualizzare e gestire le immagini generate

Le immagini generate vengono visualizzate direttamente nella chat. Puoi:

  • Cliccare sull'immagine per ingrandirla
  • Fare clic destro per salvarla sul tuo dispositivo
  • Rivedere le immagini nella cronologia della chat

7. Impostazioni e Profilo

Gestire le informazioni dell'account

Le informazioni del tuo account sono visualizzate nella dashboard e non possono essere modificate direttamente dall'interfaccia.

Cambiare la password

Attualmente, per cambiare la password, segui questi passaggi:

  1. Effettua il logout dal tuo account
  2. Clicca su "Hai dimenticato la password?" nella pagina di login
  3. Segui il processo di reset password

Resettare la password dimenticata

  1. Vai alla pagina di login
  2. Clicca su "Hai dimenticato la password?"
  3. Inserisci la tua email
  4. Segui le istruzioni inviate via email

Consigli per la sicurezza dell'account

  • Usa una password forte e unica
  • Non condividere la tua API Key
  • Effettua il logout da dispositivi condivisi
  • Mantieni il tuo browser aggiornato

8. FAQ e Risoluzione Problemi

La mia API Key non funziona +

Verifica che:

  • La chiave sia stata copiata correttamente
  • Il tuo account OpenAI abbia credito sufficiente
  • La chiave non sia stata revocata
La generazione di immagini non funziona +

Controlla che:

  • Non abbia superato il limite giornaliero di immagini
  • La tua API Key sia configurata correttamente
  • Il prompt non sia troppo lungo o ambiguo
La cronologia delle conversazioni è mancante +

Prova a:

  • Ricaricare la pagina
  • Verificare la tua connessione internet
  • Contattare il supporto se il problema persiste
Errori di rate limiting e quota superata +

Questi errori si verificano quando:

  • Hai superato il limite di messaggi mensili
  • Hai superato il limite di immagini giornaliere
  • Il tuo account OpenAI ha raggiunto i limiti di utilizzo

Soluzione: Aggiorna il tuo piano o attendi il reset dei limiti.

Problemi di caching del browser +

Prova a:

  • Premere Ctrl+F5 per ricaricare senza cache
  • Cancellare la cache del browser
  • Provare un browser diverso

9. Consigli e Best Practices

Scrivere personalità efficaci per i bot

  • Sii specifico sul ruolo del bot
  • Definisci il tono e lo stile della comunicazione
  • Includi esempi di comportamenti desiderati
  • Specifica limiti e vincoli
Esempio: "Sei un assistente esperto in cucina italiana, specializzato in ricette regionali. Rispondi sempre in modo amichevole e professionale, fornendo dettagli precisi su ingredienti e procedure."

Strutturare conversazioni multi-turno

  • Inizia con una domanda chiara
  • Mantieni il contesto tra i messaggi
  • Usa riferimenti ai messaggi precedenti
  • Sii paziente per risposte complesse

Gestione della memoria

  • Ricorda che la memoria è limitata dal tuo piano
  • Per conversazioni lunghe, riassumi periodicamente
  • Usa nomi e riferimenti chiari per mantenere il contesto

Considerazioni sulla privacy

  • Non condividere informazioni personali sensibili
  • Ricorda che le conversazioni vengono salvate
  • Usa password sicure per il tuo account
  • Non condividere il tuo account con altri
